Informática, Tecnologías web

OBJETIVO

Reconocer y aprender las metodologías para el uso de tecnologías web, aprovechandolas para combatir la crisis alimentaria.

Tecnologías web, HTML, CSS y JS

En el área de informática se llevaron a cabo temas como la tecnología web, estas incluyen HTML5, CSS y JS, estas tecnologías son lenguajes de programación que constituyen el codigo fuente de todas las páginas web en el internet, las que usamos para navegar, consultar información, ver videos, imagenes, etc.

Los lenguajes de programación más comunes son los siguientes:

● HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear páginas web. Se utiliza para definir la estructura y el contenido de una página web, como el encabezado, el cuerpo, los enlaces, las imágenes, los videos, los formularios, etc. HTML se compone de etiquetas que se utilizan para describir la estructura y el contenido de la página web.

● CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para dar estilo y presentación a páginas web creadas en HTML. CSS permite separar la presentación visual de una página web del contenido, lo que permite un diseño más eficiente y flexible. Se utiliza para aplicar estilos visuales como el color, el tamaño, el tipo de letra, el margen, el relleno, el fondo, la animación, etc.

● JS (JavaScript) es un lenguaje de programación utilizado para crear interactividad en las páginas web. Permite añadir interacciones como efectos de desplazamiento, animaciones, validaciones de formularios y otros comportamientos dinámicos en una página web. Además, se puede utilizar para acceder a datos, consumir APIs y crear aplicaciones web más complejas.

Como usar estas tecnologias en la Crisis Alimentaria

Estas tecnologías se pueden utilizar para abordar la crisis alimentaria de diferentes maneras.

Por ejemplo, HTML se puede utilizar para crear sitios web informativos sobre la crisis alimentaria y cómo abordarla, o para crear aplicaciones web que ayuden a las personas a encontrar alimentos saludables y asequibles en su área.

CSS se puede utilizar para diseñar interfaces de usuario atractivas y accesibles para estas aplicaciones y sitios web, mientras que JS se puede utilizar para agregar interactividad y mejorar la experiencia del usuario.

Además, estas tecnologías se pueden utilizar en sistemas de monitoreo y control de cultivos hidropónicos, en la creación de plataformas de intercambio de alimentos y en la automatización de procesos de producción de alimentos.

En resumen, HTML, CSS y JS pueden ser herramientas poderosas para abordar la crisis alimentaria desde diferentes ángulos y en diferentes niveles.
